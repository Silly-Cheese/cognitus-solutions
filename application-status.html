Applicants can use this page to view the latest known status of their Cognitus Solutions application using their Discord username.

Status workflow

Applications are intended to move through a structured flow such as submitted, under review, approved, or denied as the full platform expands.

Public visibility

Public applicants should only be able to view their own application information and not the internal review data of other users.

In the Firestore-connected version, this page will read the latest record directly from the applications collection and respect protected access rules.
